## Stage 1: INIT - Load References

Start the Mapping Assistant to load the Senzing specification and examples into Q's context.

Open Amazon Q Developer (if not already open)

**Tell Amazon Q:** `Start the senzing mapping assistant`

![INIT stage - Loading references](/images/exercise1/3-mapping1.png)

The AI will:
- Load the Senzing entity specification (2,100+ lines)
- Review mapping examples
- Verify the JSON linter tool is available

When complete, Q will confirm it's ready to receive your schema.

{{% notice info %}}
**Checkpoint:** The AI should confirm it has loaded the Senzing specification and is ready to proceed to the INVENTORY stage.
{{% /notice %}}
