# (Optional) Additional Languages

This page covers optional setup for additional programming languages that may enhance your workshop experience.

## Overview

While the core workshop can be completed with the default setup, you may want to configure additional language support for:
- Enhanced code completion
- Better syntax highlighting
- Language-specific debugging tools

## Supported Languages

### Python
- Install Python runtime and pip
- Configure virtual environments
- Install workshop-specific Python packages

### Node.js
- Install Node.js and npm
- Set up package management
- Configure development dependencies

### Java
- Install JDK
- Configure build tools (Maven/Gradle)
- Set up IDE integration

### Go
- Install Go runtime
- Configure GOPATH and modules
- Set up development tools

## Installation Steps

### For Local Development

1. **Choose Your Languages**
   - Identify which additional languages you need
   - Check compatibility with your operating system

2. **Install Runtimes**
   - Download and install language runtimes
   - Configure PATH variables
   - Verify installations

3. **IDE Configuration**
   - Install language extensions
   - Configure syntax highlighting
   - Set up debugging support

### For Cloud-Based IDEs

Most cloud IDEs come with multiple languages pre-installed. Check your IDE documentation for:
- Available language runtimes
- Extension marketplace
- Configuration options

{{% notice tip %}}
You can complete the workshop without additional languages, but they may provide a richer development experience.
{{% /notice %}}

## Next Steps

With your environment fully configured, you're ready to begin the main workshop modules.
